# Tema Kasus

Tema yang dipilih adalah _Sistem Perpustakaan Sederhana_.
Aplikasi ini merupakan sistem perpustakaan sederhana berbasis web
yang digunakan untuk mengelola data buku (menampilkan, menambah, mengubah dan menghapus).

# Fitur

1. Login dan Logout (menggunakan Session)
2. Password dienkripsi menggunakan password_hash()
3. Tambah buku, edit buku, hapus buku dan tampil data buku.

# Database

Menggunakan 2 tabel:

1. users (untuk login)
2. buku (untuk data buku)
